Autonomous Mobile Robot

Path Planning and Obstacle Avoidance using Dijkstra's Algorithm

Introduction

The project revolves around the design and implementation of an Autonomous Mobile Robot (AMR) capable of path planning and obstacle avoidance in a static environment. This system is engineered to operate autonomously, making decisions based on real-time data from its sensors and pre-programmed algorithms. The robot's primary functions—pathfinding and obstacle avoidance—are achieved through a combination of Dijkstra's algorithm and RFID-based navigation.

Key Aspects

1. Path Planning

Path planning is crucial for any autonomous system that needs to move between two points without human intervention. The robot uses Dijkstra’s algorithm to calculate the shortest path from its starting position to the desired destination, avoiding obstacles along the way. The algorithm ensures that the robot can navigate efficiently by considering multiple factors such as distance, time, and computational cost. By continuously recalculating the path, it adapts to changing conditions in its environment.

Effective path planning ensures that the robot can move autonomously without human input, while also adapting to changing environments. It minimizes detours or unnecessary complexity, allowing the robot to reach its goal efficiently.

2. Obstacle Avoidance

Obstacle avoidance is a vital aspect of any autonomous navigation system. In this project, the robot utilizes real-time feedback from its RFID sensors to identify obstacles and dynamically adjust its route. The system is designed to detect and circumvent any objects that may obstruct its path, whether static or movable. By using RFID technology, the robot can navigate a 3x3 grid environment and make precise movements based on the unique RFID tags placed in the grid.

3. RFID-Based Navigation

RFID (Radio Frequency Identification) technology is integral to the robot's navigation. Each position in the environment is marked with an RFID tag that contains a unique identifier (UID). The robot's RFID reader scans these tags as it moves, enabling it to track its current position and determine the next waypoint based on the calculated path. This setup allows for precise navigation, especially in environments with fixed obstacles.

Methodology

The methodology for this project focuses on developing a system for autonomous mobile robots to achieve reliable path planning and obstacle avoidance in static environments. The approach begins with selecting appropriate algorithms for path planning, such as Dijkstra's algorithm. Sensors such as RFID readers guide the robot through its environment, and the movement is controlled based on the algorithm's output.

This methodology aims to balance computational efficiency with adaptability, ensuring safe and optimal navigation for the autonomous robot in diverse conditions.

Working Mechanism of Robot

Our project centers around the design and implementation of an autonomous robot capable of path planning and obstacle avoidance. The robot uses an Arduino Uno R3, RFID card reader, and Johnson motors, and is powered by a 12V supply. The RFID system enables precise navigation by scanning pre-programmed RFID tags in a 3x3 matrix grid, guiding the robot to its destination.

Experimental Setup

1.1 Environment Setup

The operational area for the robot is a 3x3 grid, each cell of which has a unique RFID tag. The grid is used for navigation, with each tag corresponding to a specific coordinate in the matrix.

1.2 Components Used

  • Arduino Uno R3 - Acts as the central controller for the robot.
  • RFID Card Reader - Captures RFID tag IDs for accurate navigation.
  • Johnson Motors - Controls the robot's wheels and movement.
  • L298N Motor Driver - Manages motor speed and direction based on PWM signals.
