ARDUINO UNO I/O EXPANSION BOARD USING I2C PROTOCOL

Efficient Expansion of Arduino's I/O Capabilities through Multiple PCF8574 Modules

Introduction

This project focuses on enhancing the capabilities of an Arduino Uno by incorporating multiple PCF8574 I/O expander modules. The Arduino Uno, with its limited 14 digital I/O pins, is expanded by adding eight additional I/O pins per module, allowing for a greater range of interfacing with external devices. This system aims to facilitate effective management and control of various input and output devices, overcoming the limitations of the Arduino Uno’s native I/O capacity.

Project Components

  • Input Board (3 Modules): Utilizes PCF8574 modules to interface with DIP switches, increasing the number of digital inputs.
  • Output Board (3 Modules): Controls LEDs, offering expanded options for visual output and control.
  • LCD Display (2 Modules with Potentiometer):LCD Enables connection and control of LCD displays for user interaction and visual representation.

Objective

The goal is to design, implement, and evaluate a system that accommodates multiple PCF8574 I/O expanders, enabling the Arduino Uno to manage a wide range of external devices, especially for applications with extensive I/O requirements.

Circuit Design

The circuit design involves integrating eight PCF8574 I/O expander modules with the Arduino Uno via the I2C bus. Each PCF8574 module provides eight additional I/O ports, resulting in a total of 64 additional I/O points. The circuit diagrams and schematics include:

Block Diagram of the 3 modules:

  • Digital Output Board: Each PCF8574 module is connected to eight LEDs with current-limiting resistors.
  • Digital Input Board: Each PCF8574 module is connected to a dip switch with 8 inbuilt switches with pull-up resistors.
  • LCD Connections: Two 16x2 LCD displays are connected to the PCF8574 modules for data display.
  • Original Board: This module will contain LEDs and dip switches to show that the existing I/O pins of the Arduino Uno board work efficiently even after expansion.

Circuit Diagram

Experimental Setup and Results

The experimental setup consists of assembling the Arduino Uno, the I/O expansion board, and the testing components on a breadboard. The experimental protocol includes:

  • Connecting the Arduino Uno to the PCF8574 modules via the I2C bus.
  • Configuring the PCF8574 modules with unique addresses.
  • Writing and uploading test code using Arduino IDE to the Arduino Uno to toggle outputs and read inputs.
  • Observing and recording the behavior of LEDs, dip switches, and LCD displays.

Experimental Results

The project successfully expanded the input/output (I/O) pin count of the Arduino Uno from 14 to 78, significantly enhancing the system's versatility and capability. This expansion allows the system to connect a greater number of external devices and sensors, essential for handling more complex tasks and larger projects. The additional I/O pins support enhanced functionality, such as:

  • Digital Output Control: Three modules are used to control LEDs, providing visual indicators of the system's operations. The LEDs on the Digital Output Board reliably illuminate, confirming the correct operation of the digital output functionality.
  • Digital Input Monitoring: Three modules monitor the state of switches. The system responds to user inputs by displaying the corresponding status of the switches on the serial monitor. Additionally, the LCD displays pertinent messages corresponding to these inputs, demonstrating the system’s ability to interpret digital inputs accurately.

This system’s expanded I/O functionality not only validates the effectiveness of the expansion but also confirms its capability to handle more complex tasks with enhanced control over external devices and sensors.
